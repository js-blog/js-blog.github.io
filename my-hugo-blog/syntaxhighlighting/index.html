语法高亮的修饰

如前所述，Hugo Learn 主题的代码语法高亮采用的是 highlight.js 库，最大的问题是不支持行号。 Hugo 本身是可以渲染代码语法高亮的，因此，将代码高亮的方案改回到官方的 Chroma。

Hugo 主题的叠加与合并

Hugo 支持多个主题的叠加与合并。 在 themes 文件夹中的每个主题都包括了位于其下的 static, layouts (templates)，和 archetypes 等文件夹。Hugo 将依照顺序合并多个主题的文件以形成最终样式。根文件夹下的 static, layouts (templates)，以及 archetypes 不在 themes 文件夹中，但却有最高的优先级。因此，将主题文件夹中的样式文件复制到根文件夹的对应位置来，再修改其中的内容，就可以覆盖掉原主题文件，从而达到修改主题样式的目的。这样做的好处是不破坏原主题文件，改乱了还原也比较方便。

/* Remove the symbols before task lists */

.task-list label {font-weight: lighter}
.task-list li {list-style-type: none}

/* To overwrite the original theme's theme.css */

.chroma code {color: #3b2a2a} /* Overwrite theme.css::650: pre code {color: whitesmoke} */
.chroma .lntable {background-color: white} 
.lntd:first-child {width: 1px}
.lntd:first-child pre {padding: 1rem 0 1rem 1rem; border-radius: 2px 0 0 2px}
.lntd pre {border-radius: 0 2px 2px 0}
span.lnt {width: 2rem; text-align: right; display: inline-block}
.chroma span.lnt {padding: 0}
td.lntd {width: 100%}
